# Information
The project is example for application using GraphQL and Nodejs for server.
Client using Angular, React.

# How to run application

- Server 

```cmd
cd .\server
npm install -g nodemon
nodemon app
```

- Client
    - React
    ```cmd
        cd .\react-client\
        npm start
    ```
    - Angular
    ```cmd
        ng serve
    ```
